## Required reading

<Steps>
<Step title="Back up your current Firefox profile">
Create a [backup profile](https://github.com/yokoffing/Betterfox/wiki/Backup).
</Step>

<Step title="Download user.js">
Download `user.js` from [here](https://raw.githubusercontent.com/yokoffing/Betterfox/main/user.js) (right click and select `Save Link Asâ€¦`).
</Step>

<Step title="Review optional config docs">
Review [Common Overrides](https://github.com/yokoffing/Betterfox/wiki/Common-Overrides) and [Optional Hardening](https://github.com/yokoffing/Betterfox/wiki/Optional-Hardening), then apply required changes.
</Step>

<Step title="Open your profile directory">
Open Firefox. In the URL bar, enter `about:profiles` and press **Enter**. For the profile to use, click **Open Folder** in **Root Directory**.
</Step>

<Step title="Place user.js and restart">
Move `user.js` into that folder and restart Firefox.
</Step>
</Steps>

After restarting Firefox:

<Steps>
<Step title="Install an ad blocker">
Install an ad blocker such as [uBlock Origin](https://addons.mozilla.org/blog/ublock-origin-everything-you-need-to-know-about-the-ad-blocker/) and apply the [recommended filters](https://github.com/yokoffing/filterlists#guidelines).
</Step>

<Step title="Enable secure DNS">
Enable DNS-level protection with a preferred [provider](https://github.com/yokoffing/Betterfox/wiki/Optional-Hardening#secure-dns).
</Step>
</Steps>

## Simple goals

1. **Minimalism:** remove features that are not needed.
2. **Efficiency:** improve Firefox speed and performance.
3. **Privacy:** protect data without breaking websites.

## Simple configs

`Fastfox`, `Securefox`, `Peskyfox`, and `Smoothfox` are guides for Firefox settings.

The `user.js` file is curated from these guides.

| List | Description |
| --- | --- |
| [Fastfox](https://github.com/yokoffing/Betterfox/blob/main/Fastfox.js) | Increase Firefox browsing speed. |
| [Securefox](https://github.com/yokoffing/Betterfox/blob/main/Securefox.js) | Protect data without causing site breakage. |
| [Peskyfox](https://github.com/yokoffing/Betterfox/blob/main/Peskyfox.js) | Reduce distractions for a cleaner browsing experience. |
| [Smoothfox](https://github.com/yokoffing/Betterfox/blob/main/Smoothfox.js) | Improve scrolling behavior with customizable options. |
| [user.js](https://github.com/yokoffing/Betterfox/blob/main/user.js) | Combined baseline config with practical defaults. |
