This page lists apps for daily work, networking, monitoring, and self-management.

## Web

| Application | Description |
| ----------- | ----------- |
| [Helium](https://helium.computer/) | Best privacy and unbiased ad-blocking by default. Handy features like native !bangs and split view. No adware, no bloat, no noise. Made for people, by people. Fully open source. |
| [Firefox](https://www.mozilla.org/en-US/) | Fast browser for Windows, macOS, Linux, and mobile. Tune with [Betterfox](https://github.com/TeamDominant/Betterfox) for stronger defaults. |
| [Betterfox](https://github.com/TeamDominant/Betterfox) | Tuned `user.js` profile for speed, privacy, and security. Based on [yokoffing/Betterfox](https://github.com/yokoffing/Betterfox) with personal improvements. |
| [Zen](https://github.com/zen-browser/desktop) | Firefox-based browser with a modern interface and privacy-focused defaults. Tune with [Betterfox](https://github.com/TeamDominant/Betterfox/zen) for stronger defaults. |
| [Floorp](https://github.com/Floorp-Projects/Floorp) | Fast, customizable browser with Betterfox-style performance and privacy improvements. |
