## Web

| Application | Description |
| ----------- | ----------- |
| [Helium](https://helium.computer/) | Приватный браузер с нейтральной встроенной блокировкой рекламы. Удобные функции вроде нативных `!bangs` и split view. Без adware и лишнего шума. Полностью open source. |
| [Firefox](https://www.mozilla.org/en-US/) | Быстрый браузер для Windows, macOS, Linux и мобильных устройств. Можно усилить через [Betterfox](https://github.com/TeamDominant/Betterfox). |
| [Betterfox](https://github.com/TeamDominant/Betterfox) | Настроенный `user.js` профиль для скорости, приватности и безопасности. Основан на [yokoffing/Betterfox](https://github.com/yokoffing/Betterfox) с личными правками. |
| [Zen](https://github.com/zen-browser/desktop) | Браузер на базе Firefox с современным интерфейсом и акцентом на приватность. Можно усилить через [Betterfox](https://github.com/TeamDominant/Betterfox/zen). |
| [Floorp](https://github.com/Floorp-Projects/Floorp) | Быстрый и настраиваемый браузер с улучшениями в стиле Betterfox. |

## Monitoring and system info

| Application | Description |
| ----------- | ----------- |

| [CPU-G](https://github.com/atareao/cpu-g) | Open-source инструмент системной информации, похожий на CPU-Z. |
